# Manual de Usuario: PROD-SYS (Guía Paso a Paso)

¡Bienvenido a PROD-SYS! Esta guía te ayudará a navegar y utilizar el sistema de trazabilidad industrial sin necesidad de términos técnicos complicados.

---

## 🏁 1. Primera Vez: Configuración Inicial (Bootstrap)

Cuando el sistema se instala por primera vez, verás una pantalla de **Inicialización**.

### ¿Qué es esto?
Es el proceso para crear al **primer administrador real** de la planta.
1.  **Datos Personales:** Ingresa tu nombre, apellido y correo corporativo.
2.  **Código Interno:** Este será tu **nombre de usuario** para entrar al sistema (ej. tu número de empleado).
3.  **Área:** Selecciona el departamento al que perteneces (ej. Administración).
4.  **Contraseña:** Crea una contraseña segura.

Una vez que guardes estos datos, el sistema se "activará" para todos los demás. **Este paso solo se hace una vez.**

---

## 🔑 2. Entrar al Sistema (Login)

1.  Usa tu **Código Interno** y la **Contraseña** que creaste o que te asignó el administrador.
2.  Si es tu primera vez, el sistema podría pedirte que cambies la contraseña por seguridad.
3.  **Dato Clave:** Si intentas entrar muchas veces con la contraseña equivocada, tu cuenta se bloqueará automáticamente para proteger tu información. Deberás contactar a un supervisor para desbloquearla.

---

## 📝 3. El Trabajo Diario: La Bitácora de Turno

La **Bitácora** es como el "diario de la fábrica". Cada vez que empieza un turno de 8 horas, se debe usar la bitácora para registrar lo que pasa.

### Abrir Turno
El inspector de turno hace clic en **"Abrir Bitácora"**. A partir de este momento, el sistema sabe quién está a cargo y en qué horario estamos.

### Registrar Producción e Incidentes
Durante el turno, se deben registrar dos cosas principales:
- **Producción:** ¿Cuántos kilos o metros se fabricaron? ¿Hubo merma (desperdicio)?
- **Incidentes:** ¿Se detuvo una máquina? ¿Faltó material? ¿Hubo un problema eléctrico? Cada incidente debe explicarse brevemente.

---

## 🏁 4. Terminar el Día: Cerrar el Turno

Cerrar el turno es el paso más importante para que la información sea válida.

1.  Haz clic en **"Cerrar Turno"**.
2.  **Si todo está bien:** El turno se cerrará y la información se guardará permanentemente.
3.  **Si falta algo (Revisión):** El sistema te avisará si hay problemas. Por ejemplo, si una máquina trabajó pero no registraste quién la operaba. Deberás escribir una **"Observación de Cierre"** explicando por qué pasó eso antes de que el sistema te permita terminar.

---

## 👥 5. Personal y Grupos

El sistema organiza a la gente en **Grupos (A, B, C)**.
- Los grupos rotan de horario cada semana (Mañana, Tarde, Noche).
- Tú no tienes que cambiar tu horario manualmente; el sistema sabe en qué turno te toca según el grupo al que perteneces.

---

## 🛑 6. ¿Qué pasa si me voy de la empresa? (Baja Lógica)

En PROD-SYS, **nada se borra**. Si un trabajador se va, se le marca como **"Baja lógica"**.
- Ya no podrá entrar al sistema.
- Sus registros de producción pasados seguirán ahí para siempre.
- Es una medida de seguridad para que la historia de la planta nunca se pierda.

---

## 💡 Consejos de Seguridad
- **Nunca compartas tu contraseña.** Cada registro en el sistema lleva tu firma digital.
- **Cierra tu sesión** si vas a dejar la computadora sola.
- **Revisa los datos antes de guardar.** Una vez cerrada la bitácora, es muy difícil corregir errores de dedo.
